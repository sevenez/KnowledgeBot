# 租房申请流程 - 张一 - 2024年8月

**流程名称：** 为新入职专家提供临时住房
**发起人：** 张一 (CEO)
**申请日期：** 2024-08-08

**一、申请信息**
-   **租房事由：** 为从海外聘请的首席科学家（李博士）提供6个月的过渡住房。
-   **期望区域：** 公司附近，通勤便利的小区
-   **户型要求：** 两室一厅
-   **预算范围：** 月租金人民币12,000元以内
-   **期望起租日：** 2024-09-01

**二、流程步骤**
1.  **发起：** 张一提出租房需求及预算。
2.  **审批：** 【自我审批】通过。
3.  **执行：** 行政部（张四）根据要求寻找房源，法务部（张二十）审核租赁合同。
    -   **执行人：** 张四、张二十
    -   **状态：** 【已完成】。房源已确定，合同已签署。

**三、后续流程**
租金由公司直接支付。
