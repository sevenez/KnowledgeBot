# 费用报销流程 - 张一 - 2024年7月

**流程名称：** CEO月度费用报销
**发起人：** 张一 (CEO)
**所属部门：** 公司总部
**申请日期：** 2024-07-31

**一、流程概述**
本流程适用于CEO张一的日常业务费用报销，由本人确认后，财务部经理直接审核付款。

**二、流程步骤**
1.  **发起：** 张一汇总本月费用票据，填写电子报销单。
2.  **审批：** 由于申请人即为最高负责人，此环节为【自我审批】。
3.  **审核：** 财务部经理（张三）审核票据合规性、金额准确性及预算符合性。
    -   **审核人：** 张三
    -   **状态：** 待审核
4.  **付款：** 财务部出纳根据审核通过的报销单进行付款。
5.  **归档：** 财务部将报销单及票据归档。
