# 租房申请流程 - 张二 - 2024年7月

**流程名称：** 为外聘专家申请过渡住房
**发起人：** 张二 (人事部经理)
**申请日期：** 2024-07-20

**一、申请信息**
-   **租房事由：** 为公司从上海聘请的财务顾问王教授提供3个月过渡住房。
-   **期望区域：** 公司附近，环境安静的小区
-   **户型要求：** 一室一厅或两室一厅
-   **预算范围：** 月租金人民币10,000元以内
-   **期望起租日：** 2024-09-15

**二、流程步骤**
1.  **发起：** 张二提出租房需求及预算。
2.  **审批：** CEO（张一）审批。
    -   **审批人：** 张一
    -   **状态：** 【待审批】
3.  **执行：** 审批通过后，行政部（张四）根据要求寻找房源，法务部（张二十）审核租赁合同。

**三、后续流程**
寻得合适房源后，将发起单独的合同审批流程。
